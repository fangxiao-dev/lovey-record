# Lovey Record Terminology

用于对齐产品、设计、Pencil MCP 与后续实现阶段的命名。当前只收敛两类术语：功能术语、UI 术语。

## 功能术语

- 爱情记录空间: 产品的整体概念，用于承载多个记录模块，而不只服务单一经期工具场景。
- 模块实例: 用户实际访问和维护的一份模块数据实体；不同入口访问的应是同一份实例。
- 月经模块: 当前 MVP 中唯一完整实现的业务模块，也是首批设计与实现的主线模块。
- 单人优先: 产品先保证单用户可独立完成记录与查看，再扩展共享能力。
- 同实例共享: 共享语义是共同访问同一模块实例，而不是复制出第二份数据。
- 共享空间: 面向共享访问的产品入口或容器语义，用于承载已共享的模块实例。
- 模块空间: 当前合并后的模块组织页面语义，包含私人区、共享区和底部信息面板。
- 模块管理: 新导航结构下对全量模块页面的推荐命名；承接模块浏览、组织与共享状态调整。
- 工作台首页: 新导航结构下的第一入口页；用于承载最近使用模块和快速返回当前任务。
- 最近使用: 按使用历史排序的模块集合；未使用过的模块默认不在工作台首页出现。
- 批量选择保存: 通过长按与拖动路径进行日期集合选择，并在顶部右侧 `保存 / 取消` 按钮处统一提交的编辑模型。
- 单日原地编辑: 点击单个日期后，直接在首页下方编辑，而不是跳转独立详情页。
- 周期推导: 由连续 `period` 记录自动派生 cycle 结果的业务规则。
- 显式记录: 用户对某一天主动保存过的 day-state 记录，与默认空白状态相区分。
- 隐式空状态: 某日期没有持久化记录时，默认视为 `none`。

## UI 术语

- 进入页: 首批页面之一，用于产品介绍和分流到不同入口。
- 我的模块页: owner 视角下的模块列表页，用于展示可进入模块及其共享状态。
- 共享空间入口页: 共享侧入口页，强调“共享访问同一模块实例”。
- 月经模块首页: 月经模块的主工作台页面，优先回答当前状态，再支持记录和补录。
- 工作台首页: 新的第一 tab 页面，强调最近使用、快速继续和轻量总览，不等同于消息中心。
- 模块管理页: 新的第二 tab 页面，承载全量模块列表和模块组织操作。
- 底部 Tab 栏: 全局导航组件；当前推荐结构为“工作台首页 + 模块管理”。
- 模块卡片: 模块入口的卡片化表达，可出现在工作台首页或模块管理页中。
- 共享头像徽记: 显示在模块卡片角落的小圆形状态徽记；用于表达共享邀请或共享对象存在。
- 状态卡: 模块首页顶部的核心状态展示卡片，用于呈现当前状态与主次信息。
- 周期窗: 模块首页中的主要日历上下文区域，用于展示当前周期窗口。
- 月视图: 用于浏览和定位日期的辅助视图，不替代首页编辑器。
- 图例: 日历状态说明区，用于解释 `period / prediction / today / detail-recorded` 的视觉含义。
- SelectedDatePanel: 选中日期后的原地编辑面板组件名。
- BatchActionButtons: 批量选择模式下显示在 jump tabs 右侧的 `保存 / 取消` 操作组件名。
- ModuleCard: 跨页面复用的模块卡片组件名。
- SharedStateBadge: 用于表达私有、已共享等状态的文字徽记组件名。

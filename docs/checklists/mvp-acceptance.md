# MVP Acceptance

- [ ] Homepage uses the 3-week Cycle Window as the primary editor.
- [ ] Tap-based single-day editing supports `period`, `spotting`, and clearing a record.
- [ ] Long-press range selection saves default `period` day records inline.
- [ ] Derived cycle blocks come only from consecutive `period` days.
- [ ] Missing dates resolve to implicit `none`.
- [ ] `今天来了` and `今天结束了` operate on the same `day_record` model as manual edits.
- [ ] Month View remains available for browsing without becoming a separate range editor.
- [ ] Shared and private entry points resolve to the same module instance data.
- [ ] WeChat DevTools manual verification completed for the above items.

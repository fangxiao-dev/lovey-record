# Legacy Repo Freeze Note

**日期：** 2026-03-22

## 结论

本仓库从今天开始转为 legacy/reference repo，不再承担新的主线产品开发。

新的 uni-app 主线应在独立工程中推进；本仓库保留的价值是：

- 产品和设计文档
- 截图与设计参考资产
- 旧原生实现的业务规则参考
- 迁移清单与重构边界说明

## 本仓库仍然适合做什么

- 查阅已批准的设计和实现计划
- 对照截图理解信息架构和状态页
- 查看原生实现中已有的业务规则与交互意图
- 补充迁移索引、参考说明、冻结备注

## 本仓库不再适合做什么

- 继续演进原生小程序 UI
- 在原生页面层上做新的主线功能开发
- 把 native 页面结构当成后续实现基线

## 迁移给新 uni-app 工程时的原则

- 复制文档，不复制旧页面代码
- 迁移规则，不迁移 `Page({})` 组织方式
- 迁移页面结构和组件命名，不迁移 WXML/WXSS 细节实现
- 需要参考具体逻辑时，在新仓库文档中指向本仓库相关文件路径

## 冻结后维护规则

- 默认只接受文档和索引类变更
- 若需要确认 legacy 行为，可在 WeChat DevTools 中做只读验证
- 若任务实质上属于 uni-app 主线，应转到新工程处理
